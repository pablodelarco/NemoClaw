SSH to the build host and validate all prerequisites for a Packer build. Then clone the marketplace-community repo with the one-apps submodule and copy all NemoClaw build files into place.

**Step 1: Validate build host prerequisites**

```bash
ssh root@100.123.42.13 'bash -s' <<'VALIDATE'
echo "=== Build Host Validation ==="
echo "--- KVM support ---"
ls -la /dev/kvm 2>/dev/null || echo "FAIL: /dev/kvm not found"
kvm-ok 2>/dev/null || (lsmod | grep kvm || echo "FAIL: KVM modules not loaded")

echo "--- Packer ---"
packer version 2>/dev/null || echo "FAIL: packer not found"

echo "--- QEMU ---"
qemu-system-x86_64 --version 2>/dev/null || echo "FAIL: qemu-system-x86_64 not found"

echo "--- guestfs-tools ---"
which virt-sysprep 2>/dev/null || echo "FAIL: virt-sysprep not found"
which virt-sparsify 2>/dev/null || echo "FAIL: virt-sparsify not found"

echo "--- mkisofs ---"
which mkisofs 2>/dev/null || which genisoimage 2>/dev/null || echo "FAIL: mkisofs/genisoimage not found"

echo "--- RAM ---"
free -h | head -2

echo "--- Disk ---"
df -h / | tail -1

echo "--- Make ---"
make --version 2>/dev/null | head -1 || echo "FAIL: make not found"

echo "=== Validation Complete ==="
VALIDATE
```

If any tool is missing, install it:
```bash
ssh root@100.123.42.13 'apt-get update && apt-get install -y qemu-system-x86 qemu-utils guestfs-tools make genisoimage'
```

If Packer is missing or too old (< 1.10.0):
```bash
ssh root@100.123.42.13 'bash -s' <<'INSTALL_PACKER'
curl -fsSL https://releases.hashicorp.com/packer/1.15.0/packer_1.15.0_linux_amd64.zip -o /tmp/packer.zip
cd /tmp && unzip -o packer.zip && mv packer /usr/local/bin/packer && chmod +x /usr/local/bin/packer

INSTALL_PACKER
```

**Step 4: Verify Ubuntu 24.04 base image exists (or download it)**

The Packer build expects `export/ubuntu2404.qcow2` as the base image. Check if the one-apps build system has it, or download it.

```bash
ssh root@100.123.42.13 'bash -s' <<'CHECK_BASE'
cd /root/marketplace-community/apps-code/community-apps
mkdir -p export
if [ ! -f export/ubuntu2404.qcow2 ]; then
    echo "Base image not found. Building or downloading..."
    # Option 1: Build from one-apps (preferred)
```

Run the Packer build on the build host. This will take 15-45 minutes depending on network speed (NemoClaw installer downloads ~2.4GB of container images during service_install).

**Run the build:**

```bash
ssh root@100.123.42.13 'bash -s' <<'BUILD'
cd /root/marketplace-community/apps-code/community-apps/packer/nemoclaw

# Set PACKER_LOG for debugging if build fails
export PACKER_LOG=1
export PACKER_LOG_PATH="/root/packer-nemoclaw-build.log"

# Run the build
# The build has two blocks:
#   1. Context ISO generation (null source + gen_context + mkisofs)
#   2. QEMU VM build (boot Ubuntu 24.04, provision, install NemoClaw)
packer build -force .

echo "=== Build Complete ==="
ls -lh /root/marketplace-community/apps-code/community-apps/output/nemoclaw.qcow2 2>/dev/null || echo "FAIL: output image not found"
BUILD
```

**Note:** The build runs headless (headless=true in variables.pkr.hcl). If it fails with SSH timeout, check:
1. Build log: `ssh root@100.123.42.13 'cat /root/packer-nemoclaw-build.log | tail -100'`
2. Context ISO generation: Was context.sh/context.iso created?
3. KVM acceleration: Is qemu using kvm or falling back to TCG?

**If the build succeeds, verify the output image:**

```bash
ssh root@100.123.42.13 'bash -s' <<'VERIFY_IMAGE'
IMAGE="/root/marketplace-community/apps-code/community-apps/output/nemoclaw.qcow2"
echo "=== Image Info ==="
qemu-img info "$IMAGE"
echo ""
echo "=== File Size ==="
ls -lh "$IMAGE"
echo ""
echo "=== Virtual Size ==="
qemu-img info --output=json "$IMAGE" | jq '.["virtual-size"]'
VERIFY_IMAGE
```
