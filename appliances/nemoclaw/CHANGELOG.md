# Changelog

All notable changes to the NemoClaw AI Agent Security appliance will be documented in this file.

## [0.1.0-1] - 2026-03-25

### Added
- Initial release of NemoClaw AI Agent Security appliance
- Ubuntu 24.04 LTS base operating system
- Docker Engine with NVIDIA Container Toolkit
- NVIDIA driver 550-server pre-installed
- Node.js 22 LTS pre-installed
- NemoClaw CLI pre-installed
- GPU detection with graceful fallback to cloud inference
- SSH and VNC access support
- Welcome message with quick start information
- 4 GB swap space configured

### Components
- NemoClaw: 0.1.0 (alpha)
- Docker Engine: latest stable
- NVIDIA Driver: 550-server
- NVIDIA Container Toolkit: latest stable
- Node.js: 22.x LTS
